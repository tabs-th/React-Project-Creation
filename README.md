# React-Project-Creation

Project Overview: Next.js Resume Project — ISM 3232

## Project Description
This is a professional resume website built using next.js and tailwind CSS. The project demonstrates the ability to scaffold a React application, manage project directories, and apply utility-first CSS for a modern, responsive layout while maintaining a strict separation of concerns.

## How to Run Locally
Because the project is located in a subfolder, please use these steps:

1. **Navigate to the project directory:**
   ```bash
   cd nextjs-resume

2. **Install Dependencies:**
    ```bash
    npm install

3. **Run the server:**
    ```bash
    npm run dev

4. **View App:**
   Open http://localhost:3000 in browser.

Implemented Features:
- Scaffolded with Next.js app router
- Resume content (education, experience, and skills)
- Styled with Tailwind CSS utility classes
- Cleaned up default page.js and CSS

Known Issues/Limitations: 
- Layout is optimized for desktop viewports
- LinkedIn and email are placeholders
